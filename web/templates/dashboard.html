Decision Model (Deep)
The model that sets portfolio strategy and reviews trade proposals from the scout. Used for high-level reasoning — runs less frequently.

Sonnet / GPT-4o — Good balance of quality and cost.
Opus / Gemini Pro — Best reasoning for complex strategies.
Tip: Use a premium model here — it only runs once per strategy refresh and once per review, so cost is minimal.
Scout Model (Fast)
The model that runs every check-in to gather market data, analyze opportunities, and propose trades. Uses the most tokens since it calls tools repeatedly.

Haiku / Mini / Flash — Recommended. Cheap enough for frequent check-ins.
Sonnet / GPT-4o — Better analysis but 5-10x more expensive per session.
Tip: Use Haiku or GPT-4o Mini here. The scout does research — the decision model handles quality control.
Check-in Interval
How often (in minutes) the scout model checks the market and proposes trades. Lower = more responsive but higher cost.
Example: 5 min = ~78 sessions/day. At ~$0.03/session with Haiku, that's ~$2.34/day.
min
Strategy Refresh Interval
How often (in minutes) the decision model re-evaluates portfolio strategy. The scout follows this strategy between refreshes.
Example: 60 min = decision model runs ~7 times/day. Keeps strategy current without excess cost.

Advanced Settings
Max Single Position %
The largest share of your total portfolio that can go into any one stock, ETF, or options contract. Prevents the agent from concentrating too much in a single bet.
Example: Set to 15% on a $100K portfolio — the agent will never put more than $15K into one position.
Max Options %
The maximum percentage of your portfolio that can be in options contracts (calls & puts) at any time. Options are higher-risk, so this cap limits exposure.
Example: Set to 20% on a $50K portfolio — no more than $10K will be in options at once.
Min Cash Reserve %
The minimum percentage of your portfolio that must stay as uninvested cash. Ensures the agent always keeps dry powder for new opportunities or market dips.
Example: Set to 10% on a $100K portfolio — the agent always keeps at least $10K in cash.
Max Daily Loss %
The maximum portfolio loss tolerated in a single day. Once this threshold is hit, the agent stops opening new positions for the rest of the session.
Example: Set to 3% on a $100K portfolio — if losses reach $3K in one day, trading pauses until the next session.
Max Daily Investment %
The max percentage of portfolio value the agent can deploy into new positions in a single day. Spreads entries over multiple sessions to reduce timing risk.
Example: Set to 25% on a $100K portfolio — up to $25K in new positions per day.
